SaasPuppy =>
Minimal RubyOnRails
Development Environment

{ Summary => "

    This LiveCD of PuppyLinux was baked fresh for the edX.org CS169 online course on RubyOnRails to atttenuate the initial learning curve for new students who typically have little to no prior experience with virtual machines or *nix environments.

    SaasPuppy is by no means a new distro. The course software requirements are minimal. The base image is currently RaringPuppy 3.9.9.2 and its Devpack (needed for compiling RubyGems). The course requirements (Ruby 1.9.3 , Rails 3.2.16 , and Heroku 3.6) were then baked in for flavor. That's all folks! " }

 

{ Motivation => "

    The current course VM image of the full Ubuntu Precise is far overkill for the purpose of learning Rails; and at nearly 2GB, this unfortunately excludes many students with slow internet connections. Even those with sufficient bandwidth typically download it with their web-browser; ending up with incomplete or corrupted downloads and must attempt multiple times.

    Once the download is successful the student must then install and learn how to configure the VM host properly; with the added complication that the image file is wrapped in a tarball which many popular archive tools do not extract properly. These obstacles prompt many students to attempt setting up their own rails environment with little or no experience; making it much more difficult for the TAs to diagnose their troubles that will inevitably arise.

    The result is that typically the bulk of the issues that the TAs must address for the first few weeks are these mostly avoidable errors with downloading , extraction , installation , configuration , and compatibility; which are frustrating to students , time-consuming for the TAs to diagnose , and quite outside the scope of the course subject matter. This SaasPuppy LiveCD project aims to remedy most of these common issues. " }

 

{ Release Notes => "

v0.1:

    This version of SaasPuppy features a full graphical desktop environment (currently JWM) and includes everything required to complete the edX CS169 course; as will all future versions. This initial version additionally includes countless everyday applications you'd expect in an OS (web browsers , text editors , media players , word processor , image editor , screen capture/recording , and mich more). Many of these non-essentials may be removed in the future in order to further reduce the download size; but for now you get the "Full Monte". It is based on PrecisePuppyLinux; which is a minified (puppified) version of Ubuntu Precise 12.04 with full access to the Ubuntu repos so you can add any other packages that you may desire (Firefox , Skype , whatever).

The packages that have been added to the base image are:

  • ruby 1.9.3
  • rails 3.2.16
  • heroku 3.6
  • nano 2.2.6

The package that have been removed from the base image are:

  • none

Other modifications:

  • Updated Puppy and Ubuntu repository indicies
  • Updated ca-certificates (for github)
  • Cloned rottenpotatoes course project skeleton
  • customized terminal color scheme to gray on black
  • customized colored bash prompt with git branch status

DISCLAIMER: This is the initial evaluation version and should be considered private beta for now; intended only for short-term educational use by students of the edX CS169 course. SaasPuppy carries the same copyright (Barry Kauler) and licensing terms (LGPL) as Upup Raring 3.9.9.2; as only Ubuntu debs and RubyGems gems have been added to the base PuppyLinux distribution. This version contains some proprietary binary software (such as Adobe Flash Player) which will be removed from SaasPuppy soon along with other non-essentials. Of course, students may always add back anything that is desired; but the primary goal is to make this development environment as small as possible and simple to use. As with any free software; no warrantee is expressed nor implied. Use at your own risk. That being said, PuppyLinux has been around since 2005 and is consistently amoung the top 10 GNU+Linux distributions according to distrowatch.com. PuppyLinux is super fast and responsive; and is known to run on PC's as slow as 0.6 GHz and with as little as 256MB RAM.

If you encounter any problems running or installing SaasPuppy or if you have any comments or suggestions for improvements; please don't hesitate to post them to the project issue tracker on github. " }
